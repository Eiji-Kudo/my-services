# ツイート案

## 採用: パターン8

OpenAIがそのうち作ってくれるだろうと思って待ってたけど、なかなか出てこないので自分で作ることにしました。

自分の興味と知識レベルに合わせたPodcastをAIが毎日届けてくれるサービス。X連携で好みの入力も不要、アカウントを繋ぐだけ。

作り込む前にどのくらい需要あるか見たいので、気になる方はいいね・リポストで教えてください！

---

## ボツ案

### パターン1（シンプル訴求）

Xの投稿から、あなたの興味をAIが読み取り、毎日専用のPodcastを自動生成するサービスを作っています。

セットアップは1分。あとは通勤中に聴くだけ。

### パターン2（課題→解決型）

「興味ある分野の情報、追いたいけど時間がない」

Xのプロフィールを読み込ませるだけで、AIがあなた専用のPodcastを毎日生成。知識レベルに合わせた内容で、通勤中にサクッとキャッチアップできます。

### パターン3（機能フォーカス）

Xのポストから興味を抽出 → AIがあなた専用エピソードを生成 → Podcastとして毎日届く

セットアップ1分、あとは聴くだけ。パーソナライズされた音声ニュースを作っています。

### パターン4（短くバズ狙い）

Xのアカウントを繋ぐだけで、AIが「あなたの興味」に特化したPodcastを毎日届けてくれるサービス、作ってます。
