aicastのX(Twitter)投稿用カルーセル画像を、HTML+CSSで作成する。ブラウザで開いて1枚ずつスクショして投稿する。

## 構成

- ファイル構成: slide1.html, slide2.html（各スライド独立） + styles.css（共通CSS） + script.js（ハッシュ表示制御） + index.html（iframe一覧）
- スライドサイズ: 1350×1080px
- index.htmlは全スライドをiframeで縦に並べて表示
- 各スライドHTMLは単体でも開ける（styles.css, script.jsを読み込み）
- URLハッシュ(#slide1等)で単体表示対応
- 外部フォント/画像なし。アイコンはSVGまたはCSS図形

## スライド構成

### Slide 1: フック + 仕組み

上半分: テキスト + iPhoneモックアップ2台を横並び
- 左: テキストエリア
  - 見出し: あなた専用の"Podcast"を自動生成。
  - サブ: 興味に合わせてオーダーメイドで、5分に編集して届ける。
- 右: iPhoneモックアップ2台
  - iPhone 1: X Import画面
    - 𝕏タイムライン・お気に入りから吹き出し風にトピックを抽出しているイメージ
      - 例: 「Claude Code skills 実運用ケーススタディ」@indygreg
      - 例: 「AI時代のエンジニアリングマネジメント」@lethain
      - 例: 「テックリードのためのデータパイプライン構築手法」@emilybache
  - iPhone 2: Now Playing画面（アルバムアート、プログレスバー、再生コントロール）

下半分: 3ステップ横並びフロー
- STEP 01「興味を取り込む」: X連携アイコン + チップ(𝕏 Feed, Claude Code, Infra)
- STEP 02「情報を集めて編集」: フィルターアイコン + チップ(15ソース → 3トピック)
- STEP 03「Podcastで届く」: ヘッドホンアイコン + 波形ビジュアライゼーション

最下部: ボトムロー
- 「寝る前 / 散歩 / 通勤で、読む代わりに追いつく。」
- バッジ: 「レベル調整（初心者〜実務）」「要点→背景→次アクション」
