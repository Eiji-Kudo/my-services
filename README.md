# my-services

A collection of personal services and tools.

## Projects

### ai-cast

AI-powered podcast generation and landing page.

- **podcast-gen** — CLI tool that generates podcast audio and thumbnail images using Gemini API
  - `npm run audio` — Generate podcast audio from a script (Gemini TTS)
  - `npm run image` — Generate a thumbnail image (Gemini Image)
  - Tech: TypeScript, tsx, Google Gemini API
- **lp** — Landing page for ai-cast
  - Tech: React 19, TanStack Router, Tailwind CSS 4, Vite
